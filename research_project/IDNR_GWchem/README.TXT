Ambient Ground Water Chemistry CD-ROM
IDNR-DOW Catalog #108

This dataset is presented in two formats: 
	- raw data (spreadsheet format)
	- ESRI ArcView shapefiles with associated metadata files.

Contact information is included in the metadata files, if you have questions.  

Included on this disk is a self-installing zip file for ESRI's ArcExplorer v.2, which is a GIS data viewer.  ArcExplorer can be installed by executing the ae2setup.exe file in the ArcExplorer subdirectory on this disk.  The program is 5.70 MB installed.
